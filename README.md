# Number Guessing Game (C++)

A simple console-based number guessing game written in C++.

## How it works
The program generates a random number between 1 and 100.
The user tries to guess the number and gets hints if the guess is too high or too low.

## Features
- Random number generation
- User input handling
- Simple feedback system

## How to run
Compile and run the main.cpp file using any C++ compiler.

## Language
- C++
